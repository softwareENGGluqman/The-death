# 🔌 The Afterlife 404 Error

A logical, programmatic, and philosophical response to the "marne ke baad pata chalega" (you'll find out after you die) argument. 

Built to demonstrate that when the biological hardware (brain) is destroyed, the software (consciousness/soul) has nowhere to run. Death is simply a return to the state before birth—a Dreamless Infinite Sleep.

### 🚀 Tech Stack
* Python
* Streamlit

### 🧠 Core Concept
* **Hardware:** Human Brain & Nervous System
* **Software:** Consciousness, Ability to feel pain/regret
* **Event (Death):** Hardware destroyed -> Software terminates (NULL).
* **Result:** Error 404 - Afterlife Not Found.

_"Death is nothing to us. When we exist, death is not; and when death exists, we are not." - Epicurus_
